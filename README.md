# WoW Sync App

Desktop app for syncing World of Warcraft addon folders across machines through a shared Git repository.

## v0 scope

- Electron desktop app with a polished setup UI.
- Source mode: intended to push addon updates.
- Client mode: intended to ingest updates.
- Config persistence in Electron `userData`.
- Runtime controls for manual sync and auto-sync.

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run start
```
